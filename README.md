# Personal Homepage

A fresh personal homepage project for replacing the current template and improving media performance.

## Goals

- Build a cleaner personal homepage design.
- Avoid the slow image-loading behavior in the current site.
- Keep deployment flexible so it can later replace or feed the existing `luna-shi.github.io` homepage repository.

## Status

Initial repository bootstrap only. Frontend framework, content structure, and deployment setup are intentionally left for the next step.
