ContextSwarm is a private prototype direction rather than a public repository card. The core idea is to treat a swarm run as a traceable system: agents receive bounded context, produce artifacts, and leave evidence that a judge or human can inspect.

This page avoids raw private implementation details. It exists to name the project family and connect the public notes on agent teams, task traces, and evaluation loops.
